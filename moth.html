The Client: MothDX

A lab that manufactures rapid covid test kits using LAMP assays, right here in Sydney.

The problem

They need a TGA compliant database to track their entire manufacturing process from ordering the base chemicals and stock needed for production, to getting their testing kits out to customers. And *everything* needs to be auditable

How we helped

My team at _nology worked together to build monolothic application; a Spring Boot Java + MySQL backend, linked to a React frontend for the lab staff to enter data during the manufacturing process and recall any relational data for audit purposes.

My experience

We started with the backend setup implementing CRUD for all entities necessary to get an MVP up and running. This included services, controllers and repositories for all endpoints. I was then keen to work on the front-end, as I wanted to see how everything would be linked up. We used axios to send queries to the database, placing all data received into the necessary tables, pages and modals. An interesting challenge was displaying both frontend validation errors for posting data, as well as custom errors from the backend when necessary. We also took care to implement convenient UI features such as autocomplete for data entry.
