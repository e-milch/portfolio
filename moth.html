MothDX Client Project Desc here

The Client: MothDX

A lab that manufactures rapid covid test using LAMP technology, right here in Sydney

The problem

They need a TGA compliant database to track their entire manufacturing process from ordering the base chemicals and stock needed for production, to getting their testing kits out to customers. And *everything* needs to be auditable

How we helped

My team at _nology worked together to build monolothic application; a Spring Boot Java + MySQL backend, linked to a React frontend for the lab staff to enter data during the manufacturing process and recall any relational data for audit purposes.
